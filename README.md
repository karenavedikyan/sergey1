# Landing: набор дизайнеров карточек (WB/Ozon)

Одностраничный сайт в премиальном минималистичном стиле (тёмная тема), с формой заявки и быстрым отправлением текста заявки через WhatsApp/Telegram.

## Файлы

- `index.html` — разметка страницы
- `styles.css` — стили (адаптив, типографика, тёмная схема)
- `script.js` — валидация формы + генерация текста заявки + ссылки на отправку

## Запуск локально

Можно просто открыть `index.html` в браузере, но для корректной работы некоторых функций лучше запустить локальный сервер:

```bash
python3 -m http.server 8000
```

После этого откройте в браузере `http://localhost:8000`.
