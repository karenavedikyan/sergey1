# Сайт-лендинг для набора дизайнеров карточек маркетплейсов

Одностраничный сайт для набора 4 дизайнеров карточек товаров для Wildberries, Ozon и других маркетплейсов.

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Затем откройте http://localhost:8000

## Структура

- **Hero** — заголовок, подзаголовок, CTA-кнопка
- **О проекте** — описание агентства и условия работы (40–50% дизайнеру)
- **Требования** — критерии отбора и этапы
- **Контакты** — телефон (WhatsApp/Telegram) и форма заявки

## Форма заявки

При отправке форма открывает WhatsApp с предзаполненным сообщением. Бэкенд не требуется.
